# Vue-ONE

原项目：[m.wufazhuce.com](http://m.wufazhuce.com/)

在线地址：暂无

## 本地启动

``` bash
# 安装依赖
npm install

# 热更新启动于 localhost:8080
npm run dev

# 生产环境构建
npm run build
```

> 参考资料 [ONE-API](https://github.com/zuijiaoluo/ONE-API)
